To update this web page:

a)  update index.html and any other files.
b)  git commit XX -m "note"  to commit the changes locally.
c)  git push   You will be asked for the github username and password which is in my password file.
d)  log onto tux and cd  public_html/https-github.com-CharsPANY
e) git pull  will pull the files to tux.
f)  the redirect from public_html/index.html on tux to this location will auto-update the changes on the official departmental page.  Therefore, only pulling the contents of https-github.com-CharsPANY directory into tux is necessary.
